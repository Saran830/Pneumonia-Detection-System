# AI in Medical Imaging

This repository contains projects exploring **Artificial Intelligence applications in medical imaging**, focusing on disease detection and diagnostic research.

## Projects

### 1️) Pneumonia Detection System

A deep learning project that uses **Convolutional Neural Networks (CNN)** to detect pneumonia from chest X-ray images.

Key Features:

* Image preprocessing and augmentation
* CNN-based classification
* Medical image analysis using TensorFlow/Keras

Project folder:
`pneumonia-detection-cnn`

---

### 2️) Lung Cancer Diagnosis Literature Review

A comprehensive literature review examining **AI and deep learning approaches for lung cancer diagnosis**, including:

* CNN-based detection models
* Ensemble learning techniques
* Explainable AI in healthcare
* Ethical and clinical challenges of AI deployment

Project folder:
`lung-cancer-literature-review`

---
